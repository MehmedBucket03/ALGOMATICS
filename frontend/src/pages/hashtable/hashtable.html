Understanding Hash Tables

A **hash table** is a data structure that maps keys to values for efficient lookups. It uses a hash function to compute an index in an array where the value is stored.

Hash tables provide **fast insertion, deletion, and retrieval** with an average time complexity of **O(1)**. However, collisions can occur, requiring handling strategies like chaining or open addressing.
